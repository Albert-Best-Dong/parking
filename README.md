# parking（智能停车管理）
手机客户端的制作正在学习，相比于后台管理要复杂一些，包含的主要功能为：停车位的定位（车主在停车场里与停车位置的定位）、车位引导、结算车费。给自己加油
智能停车场后台管理系统。。。。简陋版
